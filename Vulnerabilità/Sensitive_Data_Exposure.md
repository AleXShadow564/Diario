# Sensitive Data Exposure

La Sensitive Data Exposure avviene quando dati sensibili non vengono protetti adeguatamente.

## Esempi
- Password memorizzate in chiaro
- Mancato utilizzo di HTTPS

## Impatti
- Furto di credenziali
- Violazioni privacy

## Prevenzione
- Crittografia TLS/HTTPS
- Hashing sicuro delle password
- Gestione sicura delle chiavi
