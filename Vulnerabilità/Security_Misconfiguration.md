# Security Misconfiguration

La Security Misconfiguration avviene quando sistemi, server o applicazioni non sono configurati in modo sicuro.

## Esempi
- Debug mode attivo in produzione
- Credenziali di default non cambiate

## Impatti
- Accesso non autorizzato
- Esposizione dati sensibili

## Prevenzione
- Hardening dei sistemi
- Configurazioni sicure di default
- Audit e review periodici
